# Test Notebooks

Interactive Jupyter notebooks for testing and exploration.

## Notebooks

- **test_generate.ipynb**: Test generation functionality interactively
- **test_dataset.ipynb**: Test dataset loading and processing
- **test_confidence.ipynb**: Test confidence computation methods
- **baselines.ipynb**: Test baseline generation methods

## Usage

Open notebooks with Jupyter:
```bash
jupyter notebook tests/notebooks/test_generate.ipynb
```

Or launch Jupyter from the notebooks directory:
```bash
cd tests/notebooks
jupyter notebook
```

## Purpose

These notebooks provide:
- Interactive testing of core functionality
- Visual inspection of results
- Step-by-step debugging
- Examples of API usage
